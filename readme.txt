Install Nodemon // For automatically restart your server.
https://nodemon.io
$ npm install -g nodemon
How to use?
$ nodemon app.js

Install express // server
https://expressjs.com
$ yarn add express

Install Mongoose // for manage db
https://mongoosejs.com
$ yarn add mongoose

Install cors
To fix CORS (Cross-Origin Resource Sharing)
$ yarn add cors

Install formidable
For manage request form-data file, image
https://www.npmjs.com/package/formidable
$ yarn add formidable
